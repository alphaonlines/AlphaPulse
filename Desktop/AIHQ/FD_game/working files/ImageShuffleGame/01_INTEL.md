# 01_INTEL.md

## Facts & Environment

### Known Information
- Project is a sub-mission under FD_game, located in working files/ImageShuffleGame.
- Game implemented as static web app using HTML5 Canvas, JavaScript, and CSS.
- Current implementation includes 4x4 grid sliding puzzle with numbered tiles, shuffle, reset, move counter, and timer.
- No external libraries or frameworks used; pure vanilla JavaScript.

### Constraints
- Must run in modern browsers (Chrome, Firefox, Safari, Edge).
- No server-side components; client-only application.
- File size should remain minimal for quick loading.
- Code must be maintainable and extensible for future features.

### Repo Details
- Parent repo: FD_game (not a git repo currently).
- Subfolder: working files/ImageShuffleGame
- Files: index.html, style.css, game.js, and standard corp docs.
