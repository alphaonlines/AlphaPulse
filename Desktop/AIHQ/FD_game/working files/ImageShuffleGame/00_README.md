## Commander's Intent: Image Shuffle Game

### One-liner Goal
Develop a functional sliding puzzle game where players arrange shuffled image tiles to form a complete picture, enhancing user engagement and cognitive skills.

### Top Priorities
1. Implement core sliding puzzle mechanics with proper tile movement and win detection.
2. Ensure responsive design and intuitive user interface.
3. Add game features like shuffle, reset, move counter, and timer.

### Non-negotiables
- Game must be playable in modern web browsers without external dependencies.
- Code must follow clean, readable JavaScript practices.
- No sensitive data or external API calls.
